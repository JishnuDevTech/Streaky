## About
Streaky is designed to turn habit tracking into a fun, interactive experience. Users can log daily activities, monitor streaks, and stay motivated with animated progress indicators. Its simple Firebase login keeps the experience smooth and secure.

---

## Features
- Track streaks for any activity
- Add custom streaks and goals
- Colorful, animated UI for engaging experience
- Daily reminders to maintain streaks
- Firebase authentication for secure login
